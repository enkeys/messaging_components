# AMOM (Abstraction Messaging of Middleware)
## Description
Abstract classes for messaging components like broker, clients, messages, queue, topics, routes.

## Abstracts API for:
- Protocol
- Message
- Broker
- Router
- Client 
 - Sender
 - Receiver
 - Connector

## Another abstract implementation
- Queue
- Addresses
- Routing logic
    - Any-cast
    - Multi-cast
    - Mixed (multi-cast + any-cast)
